# 4 промпта для глубокого исследования запуска @ai_dlya_doma

**Контекст проекта:**
- Telegram-канал об AI-лайфхаках для дома и быта
- Полная автоматизация через бота
- Целевая аудитория: платёжеспособные женщины 25-45, AI-новички
- Инструменты: Claude ($100), ChatGPT, Midjourney
- Бюджет на рекламу: до $100
- Цель: монетизация рекламой через 6 месяцев

---

## ПРОМПТ 1: Техническая архитектура автоматизации

```
Ты — senior-разработчик с опытом создания автоматизированных Telegram-каналов и контент-систем. Проведи глубокое исследование и создай детальную техническую архитектуру для полностью автоматизированного Telegram-канала @ai_dlya_doma (AI-лайфхаки для дома и быта).

## Контекст
- Канал должен работать на 90%+ автопилоте
- Доступные инструменты: Claude API ($100/мес), ChatGPT API, Midjourney, n8n/Make (автоматизация)
- Публикация: 4-5 постов в день по расписанию
- Контент: AI-инструменты, промпты, лайфхаки для бытовых задач

## Задачи исследования

### 1. Источники контента
Проанализируй и составь список:
- 20+ RSS-лент и источников новостей об AI-инструментах
- 10+ Reddit/Twitter аккаунтов для мониторинга трендов
- 5+ способов автоматического обнаружения новых AI-сервисов
- Как фильтровать контент по релевантности для "бытовой" тематики

### 2. Пайплайн обработки контента
Опиши пошагово:
- Сбор сырых данных (что, откуда, как часто)
- Фильтрация по релевантности (критерии, ключевые слова)
- Генерация поста через LLM (какой промпт, какая модель)
- Генерация картинки (когда нужна, какой промпт для Midjourney)
- Модерация (автоматическая проверка качества)
- Очередь публикации (буфер, расписание)
- Публикация в Telegram

### 3. Технический стек
Сравни варианты и дай рекомендацию:
- n8n vs Make vs Zapier vs кастомный Python-скрипт
- Telegram Bot API vs сторонние сервисы (ControllerBot, etc.)
- Хостинг: где разместить (VPS, serverless, etc.)
- База данных: нужна ли, какая
- Стоимость инфраструктуры в месяц

### 4. Промпты для генерации
Создай production-ready промпты:
- Промпт для фильтрации новостей (релевантно/нерелевантно)
- Промпт для генерации поста "AI-инструмент дня"
- Промпт для генерации "Лайфхак дня"
- Промпт для генерации "Промпт недели"
- Промпт для проверки качества сгенерированного поста

### 5. Обработка ошибок
- Что делать, если источник недоступен
- Что делать, если LLM сгенерировал плохой контент
- Как избежать дублей
- Как обрабатывать rate limits API
- Мониторинг и алерты

### 6. Оптимизация стоимости
Рассчитай:
- Сколько токенов/запросов в день для 5 постов
- Claude vs GPT-4 vs GPT-3.5 — что выгоднее для каких задач
- Как уложиться в $100/мес на Claude + другие расходы
- Где можно сэкономить без потери качества

## Формат ответа
1. Архитектурная диаграмма (текстом/ASCII)
2. Пошаговый план реализации
3. Готовые промпты (copy-paste ready)
4. Оценка стоимости по компонентам
5. Чек-лист запуска технической части
```

## ПРОМПТ 2: Контент-стратегия и рубрикатор

```
Ты — контент-стратег с опытом развития Telegram-каналов от 0 до 100K+ подписчиков в русскоязычном сегменте. Проведи глубокое исследование и создай полную контент-стратегию для канала @ai_dlya_doma.

## Контекст
- Тематика: AI-инструменты и лайфхаки для дома, быта, повседневных задач
- ЦА: женщины 25-45, платёжеспособные, AI-новички, ценят время
- Контент генерируется автоматически (бот + LLM)
- Цель: монетизация рекламой через 6 месяцев
- ToV: "Практичный эксперт с человеческим лицом"

## Задачи исследования

### 1. Анализ успешных каналов
Проанализируй 15-20 каналов и извлеки паттерны:
- 5 топовых AI-каналов на русском (GPTMain, Николай про AI, etc.)
- 5 топовых каналов про быт/лайфхаки
- 5 каналов которые успешно монетизируются рекламой при 5-20K подписчиков
- Что общего у постов с высоким engagement
- Какие форматы получают больше репостов (виральность)
- Какие заголовки работают лучше всего

### 2. Рубрикатор
Разработай 7-10 рубрик:
- Название рубрики
- Эмодзи-маркер
- Описание и цель
- Частота (раз в день/неделю)
- Шаблон поста
- Пример готового поста
- Можно ли автоматизировать на 100%

### 3. Контент-микс для монетизации
Определи оптимальное соотношение:
- Полезный контент (инструменты, лайфхаки)
- Вовлекающий контент (опросы, вопросы)
- Виральный контент (подборки, чек-листы)
- Нативная реклама (как органично вписывать)
- Прямая реклама (сколько % допустимо)

### 4. Редакционный календарь
Создай шаблон на месяц:
- Расписание по дням недели
- Расписание по времени суток
- Тематические недели/месяцы
- Сезонный контент (праздники, события)
- Как планировать контент на месяц вперёд

### 5. Форматы контента
Для каждого формата опиши:
- Текстовый пост (структура, длина)
- Пост с картинкой (когда нужна, какая)
- Карточки/карусели (как создавать автоматически)
- Опросы (типы, частота)
- Видео-кружки (нужны ли, как автоматизировать)

### 6. Виральные механики
Исследуй и опиши:
- Какой контент репостят чаще всего
- Как создавать "сохраняемый" контент
- Триггеры для комментариев
- Механики вовлечения без ручной работы

### 7. A/B тестирование
План экспериментов:
- Что тестировать в первый месяц
- Как измерять результаты
- Какие метрики важны
- Как автоматизировать сбор данных

## Формат ответа
1. Таблица анализа конкурентов с метриками
2. Полный рубрикатор с примерами
3. Шаблон редакционного календаря
4. 20 готовых постов разных форматов
5. Чек-лист для оценки качества контента
```

## ПРОМПТ 3: Стратегия роста и продвижения

```
Ты — growth-маркетолог с опытом органического роста Telegram-каналов от 0 до 50K+ подписчиков с минимальным бюджетом. Проведи глубокое исследование и создай стратегию роста для @ai_dlya_doma.

## Контекст
- Новый канал, 0 подписчиков
- Бюджет на продвижение: $100 (единоразово или первый месяц)
- Нет команды, всё делает один человек
- Контент автоматизирован
- Цель: 5000+ подписчиков за 6 месяцев
- Важно: качество подписчиков > количество (для монетизации)

## Задачи исследования

### 1. Бесплатные методы продвижения
Для каждого метода опиши:
- Как работает
- Сколько времени занимает
- Ожидаемый результат (подписчиков в месяц)
- Пошаговая инструкция
- Подводные камни

Методы для анализа:
- Взаимопиар с каналами
- Комментинг в крупных каналах
- Репосты в тематические чаты
- Гостевые посты
- Коллаборации с блогерами
- Размещение в каталогах каналов
- SEO (Яндекс индексирует Telegram)
- Cross-posting в другие соцсети
- Вирусный контент

### 2. Платное продвижение на $100
Сравни варианты и дай рекомендацию:
- Telegram Ads (официальная реклама)
- Реклама в каналах (какие каналы, какая цена)
- Реклама в чатах
- Таргетированная реклама ВК/Яндекс с переходом в TG
- Посевы в тематических пабликах

Для лучшего варианта:
- Как потратить $100 максимально эффективно
- Какие каналы выбрать
- Как составить рекламный пост
- Когда размещать
- Как отслеживать эффективность

### 3. Воронка привлечения
Опиши путь пользователя:
- Где узнаёт о канале
- Что видит при переходе (закреп, описание)
- Что заставляет подписаться
- Как удержать в первые 7 дней
- Как превратить в активного подписчика

### 4. Каналы для взаимопиара
Составь список 30+ каналов:
- Название и ссылка
- Количество подписчиков
- Тематика
- Контакт админа (если публичный)
- Вероятность согласия на взаимопиар
- Приоритет (высокий/средний/низкий)

### 5. Партнёрства и коллаборации
Исследуй возможности:
- Блогеры для коллабораций
- Telegram-каналы для совместных проектов
- Бренды для бартера
- Эксперты для гостевых постов

### 6. План роста по месяцам

Месяц 1: Цель ___ подписчиков
- Действия
- Бюджет
- Метрики

(Аналогично для месяцев 2-6)

### 7. Метрики и аналитика
- Какие показатели отслеживать
- Какие инструменты использовать
- Как понять, что стратегия работает
- Когда корректировать курс

## Формат ответа
1. Рейтинг методов продвижения по ROI
2. Пошаговый план на $100 бюджета
3. Список 30+ каналов для взаимопиара
4. Шаблоны сообщений для договорённостей
5. Дорожная карта роста на 6 месяцев
6. Дашборд метрик (что отслеживать)
```

## ПРОМПТ 4: Монетизация и бизнес-модель

```
Ты — медиабайер и владелец сети Telegram-каналов с опытом монетизации каналов от 3K подписчиков. Проведи глубокое исследование и создай стратегию монетизации для @ai_dlya_doma.

## Контекст
- Канал про AI-лайфхаки для дома и быта
- ЦА: женщины 25-45, платёжеспособные
- Цель: выход на стабильный доход через 6 месяцев
- Далее: воронка в AI-агентство KLYMO (автоматизация для бизнеса)
- Важно: не испортить репутацию канала агрессивной рекламой

## Задачи исследования

### 1. Анализ рынка рекламы в Telegram
Исследуй текущие цены и тренды:
- CPM по тематикам (технологии, лайфстайл, быт)
- Средние цены за пост для каналов 5K, 10K, 20K, 50K
- Какие форматы рекламы востребованы
- Сезонность (когда больше рекламодателей)
- Тренды 2024-2025

### 2. Типы монетизации
Для каждого типа опиши:
- Как работает
- Когда можно начинать (мин. подписчиков)
- Сколько можно заработать
- Плюсы и минусы
- Как не навредить каналу

Типы:
- Прямая реклама (посты)
- Нативная реклама (интеграции)
- Партнёрские программы (CPA)
- Собственные продукты/услуги
- Донаты/подписка
- Продажа рекламы через биржи

### 3. Ценообразование
Создай калькулятор цен:
- Формула расчёта цены поста
- Как учитывать охват, вовлечённость, ЦА
- Скидки за объём
- Наценка за срочность
- Прайс-лист для разных форматов

### 4. Поиск рекламодателей
Исследуй каналы привлечения:
- Биржи рекламы (Telega.in, и др.) — условия, комиссии
- Прямые продажи — как искать, как питчить
- Рекламные агентства — стоит ли работать
- Как создать входящий поток заявок

### 5. Медиакит
Создай шаблон презентации канала:
- Структура медиакита
- Какие метрики показывать
- Как презентовать ЦА
- Кейсы размещений
- Визуальное оформление

### 6. Работа с рекламодателями
Опиши процессы:
- Первый контакт — скрипт общения
- Согласование ТЗ
- Создание рекламного поста
- Согласование и правки
- Размещение
- Отчётность
- Повторные продажи

### 7. Нативная интеграция KLYMO
Стратегия продвижения своего бизнеса:
- Когда начинать упоминать KLYMO
- Как делать это нативно
- Воронка: подписчик → лид → клиент
- Какие услуги KLYMO продвигать через канал
- Как не превратить канал в "рекламу себя"

### 8. Финансовая модель
Построй прогноз:
- Месяц 3-4: первые размещения
- Месяц 6: стабильный доход
- Месяц 12: масштабирование

Для каждого этапа:
- Подписчики
- Охват поста
- Цена поста
- Кол-во размещений в месяц
- Доход
- Расходы
- Чистая прибыль

### 9. Юридические вопросы
- Нужно ли маркировать рекламу
- Как оформлять доход (ФОП, самозанятость)
- Договоры с рекламодателями
- Налоги

### 10. Масштабирование
План на год+:
- Когда запускать второй канал
- Как автоматизировать продажи рекламы
- Создание сети каналов
- Найм помощников

## Формат ответа
1. Анализ рынка с цифрами
2. Калькулятор цен (формулы)
3. Готовый медиакит (структура + текст)
4. Скрипты продаж
5. Финансовая модель на 12 месяцев
6. Чек-лист подготовки к монетизации
```

## Как использовать эти промпты

### Порядок исследования:
1. **Промпт 2 (Контент)** — сначала понять, ЧТО публиковать
2. **Промпт 1 (Техническая часть)** — потом КАК автоматизировать
3. **Промпт 3 (Рост)** — затем КАК привлекать подписчиков
4. **Промпт 4 (Монетизация)** — и наконец КАК зарабатывать

### Рекомендации по использованию:
- Используй Claude с тарифом $100 — хватит контекста для глубокого анализа
- Каждый промпт запускай в новом чате
- Сохраняй результаты в отдельные документы
- После всех 4 исследований — сведи в единый план действий

### Ожидаемый результат:
- 4 документа по 15-30 страниц каждый
- Полная дорожная карта проекта
- Готовые к использованию промпты, шаблоны, скрипты
- Финансовая модель с прогнозами
